# Client Directory

## What It Is
A central place to manage clients, Jira mappings, hours, and portal settings.

## Why MSPs Use It
- Easy client onboarding  
- Bulk migrations via CSV  
- Generate portal links  
- Track purchased vs consumed hours  

## How to Use
Client Directory → Add/Edit clients → Configure project mapping → Save.

## Tip
Use **Bulk CSV Import** when onboarding multiple clients.
