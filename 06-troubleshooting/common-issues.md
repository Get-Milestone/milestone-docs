# Common Issues & Fixes

## 1. "Installation not found"
Cause: Installation record missing  
Fix: Reinstall app  

## 2. No Jira project mapped
Fix: Map project in Client Directory  

## 3. Permission errors
Fix: Re-approve app permissions  

## 4. Portal link expired
Fix: Generate new link  

## 5. Hours not consuming
Fix: Wait 10 mins → check worklogs → check history  

## 6. SLA timer not starting
Fix: Add comment or update status  

## 7. First user not admin
Fix: Logout → login → contact support  

## 8. Requests not syncing
Fix: Check project mapping → check link  

## 9. Hours showing 0/0
Fix: Set hours purchased  

## 10. Messages not syncing
Fix: Refresh → verify request ID
