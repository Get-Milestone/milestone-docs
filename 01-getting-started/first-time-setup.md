# First-Time Setup

Complete these steps in the first 15 minutes after install.

## 1. Add a Client
Go to Client Directory → Add Client → Enter name and identifier → Save.

## 2. Map to Jira Project
Select the client → choose a Jira project → set default issue type.

## 3. Configure SLA Defaults
Admin Console → SLA Settings → Set response & resolution times.

## 4. Generate Client Portal Link
Client Directory → Portal Links → Generate Link → Share with end customer.

## 5. Submit Test Request
Use the portal link → submit a request → confirm it appears in Unified Queue & Jira.

## 6. Invite Users
Admins: Full access  
Members: Queue, My Work, Messenger
