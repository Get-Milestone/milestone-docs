# Install from Atlassian Marketplace

Follow these steps to install the Milestone Jira App.

## Steps
1. Open Jira → Apps → Find new apps
2. Search for **Milestone**
3. Click **Install**
4. Approve requested permissions:
   - Create issues
   - Browse projects
   - Edit issues
   - View worklogs
5. Wait for installation to complete
6. The first user automatically becomes an admin

## Verification
- Go to Apps → Milestone
- You should see the Unified Queue
